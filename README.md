## 💻 Sobre o projeto

<!-- EXPLICA O MOTIVO DO PROJETO -->
HTML-Portfólio é um projeto, que foi desenvolvido para criação do meu portfólio pessoal.

O desafio foi proposto pelo bootcamp Desenvolvimento Web FullStack da professora Angela na plataforma da Udemy.

O objetivo deste projeto era criar um protótipo simples de um portfólio pessoal somente em HTML, usando tags como:

links, imagens, listas, títulos e parágrafos.
